# Android推特视频下载器
## 基本功能
- 解析推文链接，生成下载链接，自动下载推文包含的视频
- 安卓9.0下支持自动获取剪切板内容，自动进行解析后台下载。安卓10及以上版本需要复制链接至悬浮窗/通知栏中解析下载
- 支持存储视频与链接推文文案（默认路径：根目录/.savedPic）
- 支持列表流\瀑布流\tiktok流查看下载的视频
- 生日彩蛋（flutter实现，目前添加5.26和12.10两个日期）
